# REAL-TIME SPEECH TRANSLATION USING SENTIMENT ANALYSIS

# Overview

Real-Time Speech Translation Using Sentiment Analysis is a web-based application that integrates Speech-to-Text Conversion, Text-to-Speech Conversion, Language Translation, and Sentiment Analysis into a single platform. The application accepts voice, text, and emoji inputs, converts speech into text, translates the content into the selected language, analyzes the sentiment of the translated text, and generates audio output through text-to-speech technology.

The project integrates Natural Language Processing (NLP), Speech Recognition, Language Translation, Sentiment Analysis, and Web Development technologies to provide real-time translation, sentiment detection, and voice-enabled interactions in a user-friendly environment.

# Emoji Input Support

The application supports emoji input for translation, sentiment analysis, and text-to-speech conversion. Users can enter emojis directly into the input field to analyze their sentiment and generate translated text and audio output.

How to Access Emojis on Windows:

1. Click inside the text input field.
2. Press Windows Key + . (period) or Windows Key + ; (semicolon).
3. Select an emoji from the emoji keyboard.
4. The selected emoji will be inserted into the input field.

# Working Process

1. Capture voice input from the user.
2. Convert speech into text using speech recognition technology.
3. Translate the recognized text into the selected language.
4. Perform sentiment analysis on the translated text.
5. Display the translated content along with the detected sentiment.

# Installation

Clone the repository:

git clone https://github.com/Lekhanajatangi/Real-Time-Speech-Translation-using-Sentiment-Analysis.git

Navigate to the project directory:

cd Real-Time-Speech-Translation-using-Sentiment-Analysis

Install the required dependencies:

pip install -r requirements.txt

Run the application:

python app.py
