
# Real-Time-Speech-Translation-using-Sentiment-Analysis
Developed a web-based Speech Processing and Sentiment Analysis application with Speech-to-Text (STT) and Text-to-Speech (TTS) features. Implemented sentiment analysis to classify speech/text as positive, negative, or neutral. Designed a responsive interface integrating speech, text, audio, and sentiment outputs.
